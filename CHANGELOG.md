# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
а версии следуют принципам [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-06-07

### Добавлено

- определение папки `Downloads` через `pathlib`;
- подтверждение пользователя перед началом сортировки;
- категории для изображений, документов, архивов, видео, музыки и установщиков;
- категория `Others` для неизвестных расширений;
- автоматическое создание целевых папок;
- защита от перезаписи файлов с одинаковыми именами;
- обработка ошибок файловой системы без остановки всей программы;
- счётчики перемещённых и пропущенных файлов;
- итоговый отчёт после завершения сортировки;
- документация проекта и файл версии.
